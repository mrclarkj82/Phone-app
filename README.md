# Linear Equations Assignment

Static Algebra I assignment app for a class roster. The student page lets each student enter their 9-digit student ID to access multiple assignments, including 30 linear-equation problems and 15 systems-of-equations problems. Students can enter answers without seeing correctness feedback, then submit once to see a locked grade.

Note: the dashboard stores submissions in the browser running the site and now refreshes automatically across tabs in that same browser. It is useful for testing the workflow, but it is not a shared online grade database yet. Student devices need a shared backend before their submissions can appear on the teacher dashboard automatically.

The student ID gate is intended as a lightweight access check for a static GitHub Pages site. It does not replace a server-side login system.
